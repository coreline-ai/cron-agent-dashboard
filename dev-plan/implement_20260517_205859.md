## 개발 목적

`cron-agent-dashboard`에 agent별 재사용 작업 절차를 안전하게 붙일 수 있는 **Agent Skills 표준 호환 SKILL.md 기반 skill registry**를 도입한다.

핵심 방향은 새 runtime 도입이 아니라, 기존 agent instructions를 보조하는 재사용 가능한 지식/절차 모듈을 workspace/agent 단위로 관리하고 prompt에 안전하게 주입하는 것이다.

## 개발 범위

- Agent Skills 표준에 가까운 `SKILL.md` frontmatter 파서 추가
- workspace별 skill registry DB/API 추가
- agent별 skill 할당 DB/API 추가
- `always` / `trigger` / `manual` activation 지원
- prompt fence 기반 skill summary/context 주입
- run event에 실제 로드된 skill 목록 기록
- Agent detail UI에서 skill 목록/할당 관리
- 문서/API/TRD/DATA_MODEL 최소 동기화

## 제외 범위

- skill scripts 자동 실행
- MCP tool 실행 또는 MCP server 구현
- remote GitHub skill clone/update lifecycle
- skill version rollback UI
- 외부 marketplace 통합
- sandbox/RBAC 구현
