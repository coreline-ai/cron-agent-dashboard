## 잔여 리스크 / 후속 과제

- **runtime 미설치 안내**: codex/claude/gemini 셋 다 없으면 chain이 즉시 fail. Settings의 `available_runtimes`가 이미 노출되어 있으므로 새 카드에도 짧은 경고 표시(예: "감지된 runtime: 0개" → 가이드 링크).
- **샘플 이슈가 너무 길어 chain depth 초과** — canned body는 acceptance criteria 3-5줄로 한정.
- **HTTP token 모드** — 새 endpoint도 기존 token middleware 적용됨 (route group 안에 있음).
- **다른 workspace에서도 "샘플 이슈 만들기" 가 노출되는 위험** — heuristic은 dev-team 패턴(Lead + 6 worker)이 정확히 일치할 때만.
