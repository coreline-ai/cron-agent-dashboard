- v0.1 안정화 이후 남은 운영성 항목 중 즉시 효과가 큰 설정 UI, 자동 유지보수, 런타임 sanity 표시를 구현한다.

## 개발 범위
- Agent timeout/retry 설정 API 저장 누락 보강 및 Agent 상세 UI 편집 추가
- Workspace 기본 timeout 설정 UI 추가
- 자동 백업 및 run log 자동 정리 유지보수 루프 추가
- CLI runtime sanity/warning 표시 추가
- 관련 문서와 테스트 보강

## 제외 범위
- 실제 codex/claude/gemini CLI 별 신규 metrics 플래그 강제 도입
- Sub-issue API/UI, auto-chain opt-in, sandbox/RBAC
- 모델 가격표 기반 자동 비용 산정
- 대규모 프론트엔드 테스트 프레임워크 신규 도입
