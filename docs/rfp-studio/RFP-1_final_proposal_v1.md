# 한국정밀산업 코퍼레이트 사이트 글로벌 리뉴얼 — 제안서 (통합 문서)

> **발주처**: 한국정밀산업㈜ | **예산**: 1.5억 원 (VAT 별도) | **기간**: 2026.06.01 ~ 2026.11.30 (6개월)

---

## Executive Summary

- 12년된 단일 언어(한국어) 워드프레스 사이트를 **한·영·중 3개국어 글로벌 코퍼레이트 사이트**로 전면 재구축하여, 수출 비중 38%→50%+ 성장을 뒷받침하는 디지털 영업 허브로 전환합니다.
- **320개 SKU 디지털 카탈로그 + CMS 자체 운영 체계**로 마케팅팀의 콘텐츠 자율성을 확보하고, 글로벌 SEO·IR·채용 페이지 현대화로 B2B 바이어 신뢰도를 KEYENCE·Siemens 수준으로 끌어올립니다.
- Next.js 16 + Strapi 5 + AWS/Vercel 하이브리드 인프라 기반으로 **Lighthouse 90+, LCP 2.5초 미만, WCAG 2.1 AA, OWASP Top 10, GDPR** 요건을 충족하며, 일본어·베트남어 확장까지 대비한 확장 가능한 아키텍처를 제안합니다.

## 4. 기술 아키텍처 (Architect)

### 4.1 기술 스택

| 계층 | 기술 | 선정 근거 |
|---|---|---|
| **Frontend** | Next.js 16 (App Router) | RSC/ISR로 성능 + SEO 최적화, i18n 라우팅 내장 |
| **CMS** | Strapi 5 (Headless) | 비개발자 친화 Admin, 다국어 콘텐츠 모델, REST/GraphQL |
| **BFF** | Next.js API Routes / Route Handlers | CMS·검색·외부 API 통합, 캐시 제어 |
| **검색** | OpenSearch (AWS) | CJK 분석기(ICU/Nori/IK), 다국어 인덱스, 관련도 튜닝 |
| **DB** | PostgreSQL (RDS) | Strapi 기본 DB, PITR 백업, 고가용성 |
| **스토리지** | S3 + CloudFront | 제품 이미지/PDF/카탈로그 CDN 배포, 서명 URL |
| **호스팅** | Vercel (Frontend) + AWS (Backend) | Edge 렌더링 + ISR 캐시, 글로벌 CDN |
| **모니터링** | Sentry + CloudWatch + Vercel Analytics | 에러 추적 + 인프라 모니터링 + 성능 분석 |


### 4.3 다국어 아키텍처

- **URL 구조**: `/{locale}/...` — locale: `ko`, `en`, `zh`
- **콘텐츠**: Strapi i18n 플러그인으로 로케일별 콘텐츠 독립 관리
- **라우팅**: Next.js App Router `[locale]` 동적 세그먼트
- **SEO**: 페이지별 hreflang 자동 생성, 로케일별 sitemap, canonical URL
- **검색**: 로케일별 OpenSearch 인덱스 (Nori/IK/Standard 분석기)
- **확장**: 로케일 추가 시 `next.config.js` + Strapi 로케일 + OpenSearch 인덱스 추가만으로 대응

### 4.4 성능 전략

| 목표 | 방법 |
|---|---|
| LCP < 2.5s | ISR + Edge 렌더링, 이미지 next/image 최적화, 폰트 preload |
| Lighthouse 90+ | 코드 스플리팅, lazy loading, Critical CSS 인라인 |
| 글로벌 응답 속도 | CloudFront CDN (아시아·북미·유럽 PoP) |
| 검색 응답 < 300ms | OpenSearch 인덱스 최적화, 캐시 레이어 |

### 4.5 보안 설계

| 영역 | 대응 |
|---|---|
| HTTPS | 전 구간 TLS 1.3, HSTS |
| OWASP Top 10 | 입력 검증, CSRF 토큰, XSS 이스케이프, SQL Injection 방어 (ORM) |
| 인증/인가 | CMS Admin MFA + RBAC, API Key 관리 |
| 개인정보 | 수집 동의, 보관 기간 정책, 삭제 요청 프로세스 (GDPR + 개인정보보호법) |
| 파일 보안 | S3 서명 URL, 다운로드 접근 제어 |
| 의존성 | Snyk/Dependabot 자동 스캔, CI 파이프라인 통합 |
