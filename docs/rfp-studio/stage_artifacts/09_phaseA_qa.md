# QA 계획 — 한국정밀산업 글로벌 리뉴얼

## 0) QA 기준 전제

- 대상: `/ko`, `/en`, `/zh` 다국어 코퍼레이트 사이트 + CMS + 검색 + 외부 연동

- 핵심 품질 목표: LCP 2.5초 미만, Lighthouse 90+, WCAG 2.1 AA, OWASP Top 10:2025 대응, CMS 발행 후 5분 이내 반영

## 4) 회귀 테스트 시나리오 — 핵심 골든 패스 5개

### GP-01. 글로벌 바이어 제품 탐색 → 문의

1. `/en` 접속
2. 제품 카탈로그 진입
3. 카테고리/스펙 필터 적용
4. 제품 상세 확인
5. 비교함에 2개 제품 추가
6. PDF 다운로드
7. 영어 문의 폼 제출
8. Slack 알림 및 GA4 conversion 이벤트 확인

**Pass:** 제품 탐색, 비교, 다운로드, 문의 전환이 모바일/데스크탑 모두에서 중단 없이 완료된다.

---

### GP-02. CMS 제품 발행 → 웹 반영 → 검색 노출

1. CMS 에디터 계정 로그인
2. 제품 초안 작성
3. 한·영·중 번역 필드 입력
4. 제품 이미지/PDF 첨부
5. 발행
6. 5분 이내 웹 상세 페이지 확인
7. OpenSearch 검색 결과 반영 확인

**Pass:** CMS 발행 데이터가 웹, 검색, sitemap에 일관되게 반영된다.

---

### GP-03. 채용 공고 조회 → Greenhouse 지원

1. `/ko/careers` 접속
2. 공고 목록 확인
3. 상세 페이지 진입
4. 복지/문화 콘텐츠 확인
5. 지원하기 클릭
6. Greenhouse 지원 페이지 이동
7. 채용 클릭 이벤트 확인

**Pass:** 공고 정보가 최신 캐시 기준으로 표시되고 외부 ATS 지원 흐름이 정상 연결된다.

---

### GP-04. 뉴스/IR 콘텐츠 조회 → PDF 다운로드

1. `/ko/news` 또는 `/en/ir` 접속
2. 카테고리 필터 적용
3. 보도자료/재무자료 상세 진입
4. PDF 첨부 다운로드
5. 메타 태그, canonical, hreflang 확인
6. 다운로드 이벤트 수집 확인

**Pass:** IR 신뢰성, 첨부파일 접근성, SEO 메타, 이벤트 추적이 정상이다.

---

### GP-05. 다국어 통합검색

1. 한국어 검색어 입력
2. 영어 검색어 입력
3. 중국어 간체 검색어 입력
4. 제품/뉴스/페이지 혼합 결과 확인
5. no-result 검색어 입력
6. 추천/문의 유도 UI 확인

**Pass:** 3개 언어 검색 결과의 관련도와 빈 결과 처리가 기준을 충족한다.
