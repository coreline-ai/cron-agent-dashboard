## 2. 예산 · 일정 · 계약 조건

*   **예산 (1억 5,000만 원)**: 3개 국어, 320개 SKU 데이터 마이그레이션, 외부 시스템 연동을 고려할 때 **'표준' 수준의 예산**입니다. 고도화된 인터랙션이나 AI 기능을 넣기에는 타이트하므로 효율적인 리소스 배분이 핵심입니다.
*   **일정 (6개월)**: 설계 1.5M, 디자인/개발 3M, 마이그레이션/검수 1.5M으로 **안정적인 일정**입니다. 다만 320개 SKU의 데이터 정제(Cleaning) 속도가 전체 일정의 리스크입니다.
*   **계약 조건**: 1년 유지보수(월 8시간) 포함.
    *   *업계 평균 및 추정*: 대형 B2B 사이트의 경우 연간 유지보수비로 구축비의 10~15%를 책정하나, 본 RFP는 소규모 전담 대응(8시간)만 요구함. 따라서 **하자보수 기간 내 핵심 성능 유지**에 집중하고, 추가 기능 개발은 별도 계약(T&M)으로 유도하는 전략이 필요합니다.

## 4. 수주 전략 (Winning Strategy)

### 📊 SWOT 분석
*   **Strengths**: B2B 제조 도메인 이해도, Headless 기반 고성능 아키텍처 역량.
*   **Weaknesses**: 기존 워드프레스 데이터의 낮은 정제도 (마이그레이션 부담).
*   **Opportunities**: 클라이언트의 북미/유럽 확장 의지 (글로벌 표준 기술 스택 제안 적기).
*   **Threats**: 가격 중심의 경쟁사 저가 입찰 가능성 (기술/디자인 배분 55점에 집중).

### 💰 가격 정책 3옵션
1.  **합리적(Essential) - 1.3억**: RFP 기본 충실, 표준 템플릿 기반 카탈로그, 기본 SEO.
2.  **표준(Standard) - 1.5억**: **(추천)** 독자적 UX 디자인, 고도화된 제품 필터, 글로벌 CDN 및 상급 SEO, 마이그레이션 기술 지원.
3.  **프리미엄(Premium) - 1.8억**: 표준 범위 + 제품 비교 분석 대시보드(CMS 내), 인터랙션 디자인 강화, 다국어 전문 카피라이팅 지원.
