# 한국정밀산업 코퍼레이트 웹사이트 글로벌 리뉴얼 제안서

> **RFP 1줄 요약**: B2B 정밀제어 제조사의 12년 된 단일언어 워드프레스 사이트를 다국어(한·영·중) 글로벌 코퍼레이트 사이트로 전면 리뉴얼하는 6개월 프로젝트  
> **핵심 키워드**: `다국어 코퍼레이트` · `제품 카탈로그 디지털화` · `글로벌 SEO` · `헤드리스 CMS` · `B2B 산업재 UX`

## 1. Executive Summary

- 한국정밀산업의 글로벌 매출 확대(38% → 50%+ 목표) 전략에 맞춰, **키엔스·지멘스급 신뢰감**을 전달하는 코퍼레이트 사이트를 구축합니다.
- 320개 SKU 카탈로그를 **웹 네이티브 스펙 필터·비교 기능**으로 전환하여, 글로벌 바이어의 제품 탐색 → 문의 전환율을 극대화합니다.
- 한·영·중 3개 언어 동시 런칭, **i18n 확장 아키텍처**로 일본어·베트남어 추가를 무코드 수준으로 지원합니다.

## 2. 클라이언트 Needs 요약 (Sales)

### 2.1 비즈니스 배경

### 2.2 핵심 Pain Point → 해결 방향

| Pain Point | 영향 | 제안 해결책 |

| PDF 카탈로그 | 모바일 가독성·검색 불가 | 구조화된 제품 DB + 웹 카탈로그 + PDF 병행 제공 |

## 5. 기술 아키텍처 (Architect)

### 5.1 기술 스택

| 계층 | 기술 | 선정 이유 |
|---|---|---|
| **프론트엔드** | Next.js 15 (App Router) | SSG/ISR 성능, i18n 내장 |
| **CMS** | Strapi v5 (Headless) | 비개발자 친화, 다국어 플러그인 |
| **검색** | Meilisearch | 한·영·중 토큰화, 셀프호스트 |
| **스타일** | Tailwind CSS 4 | 디자인 시스템 토큰화 |
| **호스팅** | Vercel + AWS | 글로벌 Edge CDN |
| **DB** | PostgreSQL 16 (RDS) | Strapi 공식 지원 |
| **스토리지** | AWS S3 | PDF/CAD/이미지 원본 |
| **모니터링** | GA4 + Vercel Analytics | 행동 + 성능 이중화 |

### 5.2 아키텍처

```
[브라우저] → HTTPS → [Vercel Edge CDN] → [Next.js ISR]
                                              ↓ API
                                         [Strapi CMS] (ECS)
                                              ↓
                                    [PostgreSQL] + [S3] + [Meilisearch]

외부 연동: Greenhouse API · Slack Webhook · reCAPTCHA v3 · GA4
```

### 5.3 다국어 전략

| 항목 | 방식 |
|---|---|
| URL | `/ko/...`, `/en/...`, `/zh/...` |
| 콘텐츠 | Strapi i18n 플러그인 |
| UI 문자열 | next-intl (JSON) |
| SEO | hreflang 태그 + 언어별 sitemap |
| 확장 | locale 추가만으로 일·베 즉시 지원 |

### 5.4 성능 목표

| 지표 | 목표 | 방안 |
|---|---|---|
| LCP | < 2.5s | ISR + CDN + AVIF |
| FID | < 100ms | 코드스플리팅 + 서버 컴포넌트 |
| CLS | < 0.1 | 이미지 dimension 예약 |
| Lighthouse | 90+ | CI Lighthouse 게이트 |

### 5.5 보안

- HTTPS 전 구간, OWASP Top 10 대응 (CSP, XSS 필터, ORM, CSRF 토큰)
- 개인정보보호법 + GDPR: 쿠키 동의, 데이터 보존 정책
- Strapi 관리자: IP 화이트리스트 + 2FA
- 월 1회 취약점 스캔
