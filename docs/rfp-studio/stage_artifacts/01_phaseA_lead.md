## 1. RFP 1줄 요약

> B2B 정밀제조기업(매출 1,200억)의 12년된 워드프레스 사이트를 **다국어(한·영·중) 제품 카탈로그 중심 글로벌 코퍼레이트 사이트**로 6개월/1.5억 규모 전면 리뉴얼

**핵심 키워드**: `다국어(i18n)` · `제품카탈로그(320 SKU)` · `글로벌SEO` · `Headless CMS` · `B2B 코퍼레이트`

## 3. 위임

각 팀은 아래 순서로 작업을 시작해 주세요.

### @Sales — 클라이언트 니즈 분석 및 win-theme 도출

다음 항목을 분석해 주세요:

- **클라이언트 pain-point 정리**: 현재 사이트 5대 문제점을 발주처 관점에서 재해석 (비즈니스 임팩트 포함)
- **경쟁사 벤치마크**: 키엔스(KEYENCE) / 지멘스(Siemens) 코퍼레이트 사이트의 강점·차별점 분석
- **평가기준 매핑**: RFP 평가항목(기술30, 디자인25, SEO15, 일정15, 가격10, 유지보수5) 각각에서 우리가 강조할 win-theme
- **타겟 바이어 페르소나**: 글로벌 바이어(북미·유럽 확장)가 이 사이트에서 기대하는 경험

---

### @Planner — 사이트맵·IA 및 기능명세

다음 항목을 설계해 주세요:

- **사이트맵**: 전체 페이지 구조 (다국어 URL prefix `/ko`, `/en`, `/zh` 반영)
- **IA(정보구조)**: 메인, 회사소개, 제품카탈로그(320 SKU), 채용, 뉴스/IR, 문의·방문예약, 통합검색
- **기능명세 요약**:
  - 제품 카탈로그: 카테고리/스펙 필터, 비교, 제품별 다운로드
  - 채용: Greenhouse ATS 연동, 사내문화 페이지
  - 뉴스/IR: 카테고리별 보도자료·공시·재무자료
  - 문의: 다국어 폼 + reCAPTCHA + Slack 알림
  - CMS: 비개발자 마케팅팀 직접 운영
  - 검색: 한·영·중 통합검색
  - 분석: GA4 + 자체 대시보드
- **콘텐츠 마이그레이션 계획**: 기존 WP 콘텐츠(제품 PDF, 뉴스 게시글 등) 이관 범위·방식·일정

> Planner 완료 후 → @Designer, @Architect 병렬 착수

---

### @Designer — 디자인 컨셉 및 UX 제안 *(Planner 완료 대기)*

Planner의 IA/사이트맵 기반으로 다음을 제안해 주세요:

- **디자인 컨셉**: B2B 정밀산업 프리미엄 톤 (키엔스·지멘스 레퍼런스 반영)
- **무드보드**: 컬러 팔레트, 타이포그래피, 이미지 톤앤매너 (CI/BI 가이드 반영 전제)
- **핵심 5종 시안 방향**: 메인·소개·제품·채용·문의 각 페이지 레이아웃 컨셉 (데스크탑+모바일)
- **모바일 우선 반응형 전략**: 브레이크포인트, 터치 인터랙션 고려사항
- **접근성**: WCAG 2.1 AA 준수 포인트 (색상 대비, 키보드 네비게이션 등)

---

### @Architect — 기술 아키텍처 설계 *(Planner 완료 대기)*

Planner의 기능명세 기반으로 다음을 설계해 주세요:

- **기술스택 제안 + 선정 근거**: Frontend(SSR/SSG), CMS(Headless), Backend, DB, 인프라
- **다국어 아키텍처**: i18n 라우팅, 콘텐츠 관리 구조, URL 전략
- **제품 카탈로그 데이터 모델**: 320 SKU, 카테고리/스펙 필터, 비교 기능 지원 스키마
- **성능 전략**: CDN 구성, LCP < 2.5s, Lighthouse 90+ 달성 방안
- **보안**: HTTPS, OWASP Top10 대응, 개인정보보호법/GDPR 준수
- **SEO 기술 전략**: 다국어 sitemap, hreflang, 영문 키워드 노출 구조
- **통합검색**: 한·영·중 다국어 검색 엔진 방안
- **확장성**: 일본어·베트남어 추가 가능 구조

> Architect 완료 후 → @QA 착수

---

### @QA — 품질 보증 계획 *(Architect 완료 대기)*

Architect의 아키텍처 기반으로 다음을 수립해 주세요:

- **테스트 전략**: 단위·통합·E2E·성능·보안·접근성 테스트 범위
- **다국어 테스트**: 3개 언어별 콘텐츠 렌더링, URL 라우팅, 폰트·레이아웃 검증
- **성능 테스트**: LCP < 2.5s 검증 시나리오 (글로벌 리전별), Lighthouse 90+ 자동 검증
- **보안 테스트**: OWASP Top10 스캔, 개인정보 처리 검증
- **접근성 테스트**: WCAG 2.1 AA 자동/수동 검증 항목
- **UAT 시나리오**: 마케팅팀(CMS), 글로벌 바이어(제품탐색), 지원자(채용) 관점
- **품질 KPI**: 핵심 메트릭 및 합격 기준
